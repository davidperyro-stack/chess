# Chess Game 

## Features

- Standard chess rules and legal move validation
-  Play against the Stockfish chess engine
- Live analysis board with evaluation bar

## language used

- HTML5
- CSS3
- JavaScript

# AI Declaration

AI was used for:
- Fixing a bug where the chess board disappeared after adding Stockfish.
- Fixing a bug where selecting an empty square caused the game to freeze.
- Assisting with making Stockfish generate and play moves correctly.
- Assisting with adding the analysis board and evaluation bar.

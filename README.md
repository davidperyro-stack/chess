# Chess Game 
A chess project that has analysis board and a working playable bot with diffrent difficulties!!

# Why i did this?
I did this project to learn javascript (and of course HTML) And because i always loved creating things

## Features

- Standard chess rules and legal move validation
-  Play against the Stockfish chess engine
- Live analysis board with evaluation bar
- BOT with 3 difficulties! (easy, Medium and Hard!)
